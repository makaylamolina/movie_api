This is a web application for viewing horror movies. The user will be able to view information on the directors add favorite movies to their account.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Allows a user to be created /users POST None A JSON object of the new user, structured like: "Username": "", "id": "", "favoriteMovie": "" A JSON object of the new user added, structured like: "Username": "", "id": "", "favoriteMovie": ""
Returns a list of all users. /users GET None None A JSON object of each user.
Gets information on one user. /users/:Username GET :Username None A JSON object of that user.
Allows a user to change their username /users/:Username PUT :Username None A JSON object of the user.
Allows a user to add a movie to their list of favorites. /users/:Username/movies/:MovieID POST :Username, movies, :MovieID None Text confirmation that the movie has been added to their list of favorites.
Allows a user to remove a movie from their list of favorites. /user/:Username/movies/:MovieID" DELETE :Username, movies, :MovieID None Text confirmation that the movie has been removed from their list of favorites.
Allows a user to delete their account. /users/:Username DELETE :Username None Text confirmation that the account has been deleted.
Returns a list of all movies. /movies GET None None A JSON object of all movies.
Returns data about one movie. /movies/:Title GET :title None A JSON object of one movie.
Returns movies in one genre. /movies/:genre GET :genre None A JSON object of the movies in that genre.
Returns movies by one director. /movies/directors/:directorName GET :directorName None A JSON object of movies by one director.
